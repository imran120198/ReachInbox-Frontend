## Features

- Login
- Light/dark mode
- Signup/login 
- Params based API call
- Delete the Thread message(pop-up box)
- Reply the thread (pop-up box)
- Shortcut key ("R" or "r" to REPLY)
- Shortcut key ("D" or "d" to DELETE)
